Interactive Model BME 6.1: Bootstrap Values

Instructions: This model illustrates the process behind bootstrap values in evolutionary trees. Table 1 contains DNA sequences from 6 species, each one with 7 bases, shown one by one in positions 1 through 7. To sample with replacement once from the 7 positions, click the "One step" button. Or, to sample with replacement 7 times all at once, click the "Whole sequence" button. The randomly selected bases from each sequence are given in table 2. Once you have taken 7 randomly samples, the distance (i.e., the number of different letters) between each pair of these sequences in the random sample are computed in table 3. Distances are color coded from white (exactly the same) to dark orange (every letter is different). Copyright Laurie J Heyer, 2018. All rights reserved.
