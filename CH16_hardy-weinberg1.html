Interactive Model BME 16.2.1: Hardy-Weinberg Equilibrium

Instructions: This model calculates Hardy-Weinberg equilibrium genotype frequencies using the formulas explained in this BME. The formulas are given in Table 16.3, or you can remind yourself of these formulas by hovering over the third column in table 1. Enter the frequencies of each genotype in the second column of Table 1 (Calculation), and the Hardy-Weinberg frequencies will appear in the third column. Table 2 (Validation) uses the same equations, confirming the values calculated in Table 1 are in Hardy-Weinberg equilibrium. Copyright Laurie J Heyer, 2014. All rights reserved.
